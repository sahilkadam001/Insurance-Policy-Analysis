# Insurance Policy Analysis Dashboard

## Project Overview

This project analyzes insurance policy data to identify business performance, policy distribution, customer trends, and claim insights using Excel, Power BI, and SQL.

## Objectives

- Analyze policy performance
- Monitor active and inactive policies
- Evaluate claim trends
- Identify top-performing policy categories

## Tools Used

- Power BI
- Microsoft Excel
- SQL
- Power Query
- DAX

## KPIs

- Total Policies
- Active Policies
- Inactive Policies
- Active Policy %
- Total Premium
- Total Claims
- Claim Ratio

## Dashboard Features

- KPI Cards
- Monthly Policy Trend
- Policy Status Analysis
- Claim Analysis
- Customer Segmentation
- Interactive Slicers

- Power BI Dashboard (.pbix)
- Dataset (.xlsx)
- SQL Queries
